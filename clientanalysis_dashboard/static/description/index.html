This app provides a free Client Analysis Dashboard developed in Power BI. In just 3 steps you can download and run the dashboard on your own Odoo company data. The dashboard represents a customer delta sales analysis. It analyses the relative evolution of sales per customer and thus serves as a leading indicator for your sales and customer relationships.

Super Easy Connection

The connection is easily established by providing parameters in Power BI environment. A video instruction is hereunder included to show you how this is in less than a few minutes established. Furthermore, note that this App is not an Odoo module, but contains mainly a Power BI file which allows representing your Odoo data in the provided dashboard.
